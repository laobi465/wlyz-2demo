# 极策k网络验证 易语言模块

> 作者: 极策k  日期: 2026-07-21

适用于 Windows 桌面软件、游戏辅助、外挂脚本等易语言开发场景的接入指南。

易语言（EPL）为中文可视化编程语言，本模块提供两种集成方案：

1. **方案 A：原生易语言实现**（推荐）— 使用「精易模块」提供的 HTTP / 加密函数，纯易语言代码实现，无需外部 DLL。
2. **方案 B：调用 C++ DLL** — 编译 `sdk/cpp` 为 `jicek.dll`，易语言通过 DLL 命令调用。

---

### 优缺点对比

| 方面 | 方案 A（原生易语言） | 方案 B（C++ DLL） |
|---|---|---|
| 部署 | 单 .e 文件，零依赖 | 需附 jicek.dll + openssl/libcurl DLL |
| 性能 | 中等（解释执行） | 高（原生编译） |
| 维护 | 易语言源码可读 | 需重新编译 DLL |
| 跨语言 | 仅易语言 | C++ DLL 可被多语言调用 |
| 推荐场景 | 轻量集成、快速开发 | 性能敏感、加密强度要求高 |

## 安全注意事项

1. **签名密钥保护**：易语言源码易被反编译，建议将 `sign_secret` 通过服务端下发或使用 VMProtect/Themida 加壳。
2. **RSA 公钥嵌入**：可硬编码在源码中（公钥泄露无害）。
3. **卡密传输**：必须 RSA 加密，禁止明文传输。
4. **心跳防重放**：每次心跳生成新 UUID，复用会被服务端拒绝（错误码 3016）。
5. **指纹采集权限**：wmic 命令需 Windows 管理员权限，普通用户可能采集失败（建议降级为 WMI COM 调用）。
